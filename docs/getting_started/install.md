# Installation Guide

cdapython runs on:

Mac OS python versions: 3.8, 3.9, 3.10

Ubuntu python versions 3.8, 3.9, 3.10

Windows python versions: 3.8, 3.9, 3.10


## Conda + Pip install

!!! requirements

    - terminal/command line
    - python version >= 3.8 [(Install)](https://www.python.org/downloads/)
    - pip version >=
